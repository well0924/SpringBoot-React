### 💬소개

지금까지 배운 것을 기반으로 SPA 방식의 프로젝트를 처음으로 구현해보았습니다.
웹 프로그래밍의 기본 소양으로 여겨지는 게시판을 직접 만들어 보면서, 효율적인 데이터 처리와 데이터 변환 등
다양한 개발 문제에 직면하고 해결책을 찾아갔습니다.

- 참여인원: 1인
- RestApi + SPA
